# Customer_behavior
Este proyecto tiene como objetivo analizar y optimizar la interacción de los clientes con el servicio, enfocándose en aspectos clave de adquisición y rentabilidad. Se estudiará cómo los clientes usan el servicio, identificando los patrones de uso y los momentos en los que inician sus compras. Además, se medirá cuánto dinero aporta cada cliente a la compañía y cuándo los ingresos generados por estos superan el costo de adquisición (CAC). A través de este análisis, buscamos mejorar la estrategia de fidelización y optimizar la rentabilidad a largo plazo del negocio.
